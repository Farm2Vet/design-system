Section 02 / Foundations Semantics.

Where primitives become purpose. Each token below names an intention — a role, an intent, sometimes a state — and resolves down to the primitives in Section 01. Design with these names, never the raw values beneath.

05 / Spacing

Element rhythm.

Semantic margins for typography blocks. Use these in place of raw spacing primitives whenever the gap is between document elements — they guarantee the rhythm stays consistent across pages.

06 / Radius

Soft corners.

Six steps from a barely-rounded micro-corner up to a fully circular pill. Each step doubles roughly along the spacing scale — pick the one that matches the surface scale, not the visual taste of the moment.

07 / Gradient

Brand washes.

A small library of named gradients, all built from primary tokens so they age cleanly with the palette. Reach for subtle washes for surfaces; reserve the bold brand gradient for moments of marketing weight.

08 / Glass

Translucent matter.

Two glass surfaces — a light one for app-on-light, a dark one for app-on-brand — plus six blur intensities. Pair the surface with a matching border and a blur large enough to neutralise the layer beneath.
