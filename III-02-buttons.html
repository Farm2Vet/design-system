Section 03 / Components Components.

Composable surfaces built on the semantic layer. Each component declares its own narrowly-scoped tokens that resolve back to the system's intent — never to raw primitives. We start with the workhorse: the Button.

Variants by intent.

04 / Button — Switch

Switch and segmented.

A binary on/off control with a sliding thumb, plus a segmented variant for one-of-N choices — the canonical shape for a language switch. Both share the toggle variant's colour scheme and align with the button height scale (LG · 48px focus).
