Section 03 / Components Components.

Composable surfaces built on the semantic layer. Each component declares its own narrowly-scoped tokens that resolve back to the system's intent — never to raw primitives. We start with the workhorse: the Button.

02 / Button — Versions

Surface versions.

The same variant adapts to its container. Inverted surfaces invert the contrast scheme; icon-only and borderless re-shape the affordance for compact rails and inline contexts.
