Reading the wireframes

These are lofi wireframes — pure grayscale, no brand color, no real iconography. The goal is to lock in information architecture, layout rhythm, and component composition before any visual polish lands.

The admin shell (sidebar + topbar) mirrors the structure used by V-04 in the design system, so every screen here will compose cleanly against the real .app-shell + .dash-topbar pattern when this work moves to hifi.

Solid dark fills mark primary actions (Submit, Save changes); outlined neutral fills mark secondary actions; the leftmost column on the list view is reserved for bulk-select; the dashed block on Create/Edit is the add-entry affordance.
