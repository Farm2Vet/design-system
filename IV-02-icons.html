02 / Tokens

Tokens.

Icons paint in currentColor; the --icon-* token rail sets the colour the consuming component inherits. Every row resolves through a semantic, never a primitive — change the semantic and every icon downstream follows.

Token References Role Preview
Neutral
--icon-default var(--neutral-700) Default body-text icon tone.
--icon-default-subtle var(--neutral-500) De-emphasised inline icons.
--icon-default-minimal var(--neutral-400) Placeholder / disabled-adjacent.
--icon-default-disabled color-mix · neutral-900 30% Disabled state.
Inverted
--icon-inverted var(--white-base) Icons on dark surfaces.
--icon-inverted-subtle color-mix · white 70% Subdued icons on dark surfaces.

--icon-primary var(--primary-400) Brand-tone icons (resting).
--icon-primary-strong var(--primary-500) Stronger brand-tone (focused / active).
