III-20 / Components Breadcrumb.

Shows a user's location in the page hierarchy. The last item is always the current page — non-interactive, labelled aria-current="page". In the admin layout it lives inside the floating glass Navbar pill.

Anatomy

.breadcrumb
The <nav> root. Add .breadcrumb--pill for the glass pill treatment used in the Navbar.
.breadcrumb__list
Ordered <ol>. Ordered because the trail has semantic depth order.
.breadcrumb__item
Each <li>. Contains one of: link + sep, or current.
.breadcrumb__sep
The › separator. Always aria-hidden="true" — purely visual.
.breadcrumb__link
An ancestor crumb. Must have an href. Hover color: --breadcrumb-fg-hover.
.breadcrumb__current
The final, non-interactive crumb. Requires aria-current="page". Rendered semibold at --breadcrumb-fg-current.

Tokens

Breadcrumb shows depth, not tabs. The last item is always the current page (aria-current="page") and must never be a link. Use Tabs for switching between peer views of the same content. Breadcrumb is for hierarchical location only.
