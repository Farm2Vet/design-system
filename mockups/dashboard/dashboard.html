Mockup conventions

Forest theme — brand color, soft shadow and a frosted-glass header, ported from the design system. Components use canonical classes (.btn, .chip, .stat, .panel). Status uses chip variants (chip--ok / chip--run / chip--warn / chip--err / chip--idle). Running rows use a progress strip with monospace meta. Score trend is a placeholder polyline — production will use the canonical chart. Names, numbers and disease IDs are illustrative.
